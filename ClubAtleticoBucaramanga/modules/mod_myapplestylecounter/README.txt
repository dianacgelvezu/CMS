README for My Apple Style Counter for Joomla

A simple Apple Style module for counting the web page visitors.
It is entirely JavaScript and jQuery based so it
doesn't need Flash player. It has been tested and proved to be OK with
Joomla 2.5.0 and newer.

INSTALLATION INSTRUCTIONS:
==========================

- download the MyAppleStyleCounter Module based on your Joomla version.
- install the Module using the Joomla Extensions install utility.
- ENABLE the new "My Apple Style Counter Module" within the Joomla 2.5 back-end Module manager.
- Remember to assign the module to the menus you want this module to be displayed on,
  (By default module is not assigned to any of the menu links).
